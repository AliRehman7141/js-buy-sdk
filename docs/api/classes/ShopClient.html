fetchQueryProducts
(
  • query
)
Promise | Array
public

Fetches a list of products matching a specified query.

client.fetchQueryProducts({ collection_id: '336903494', tag: ['hats'] }).then(products => {
  console.log(products); // An array of products in collection '336903494' having the tag 'hats'
});

Parameters:

  • query Object

    A query sent to the api server containing one or more of:

    • [collection_id] String | Number optional

      The ID of a collection to retrieve products from

    • [tag] Array optional

      A list of tags to filter the products by. Accepts up to 10 tags.

    • [product_ids] Array optional

      A list of product IDs to retrieve

    • [page=1] String | Number optional

      The page offset number of the current lookup (based on the limit)

    • [limit=50] String | Number optional

      The number of products to retrieve per page

    • [handle] String optional

      The handle of the product to look up

    • [updated_at_min] String optional

      Products updated since the supplied timestamp (format: 2008-12-31 03:00)

    • [sort_by] String optional

      Will modify how products are ordered. Possible values are: "updated_at", "best-selling", "title-ascending", "title-descending", "price-descending", "price-ascending", "created-descending", "created-ascending", or "collection-default". Using "collection-default" means that products will be ordered the using the custom ordering defined in your Shopify Admin. Default value "collection-default".

Returns:

Promise | Array:

The product models.
