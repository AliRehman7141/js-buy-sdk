ProductVariantModel

Module: shopify-buy

Model for product variant

Constructor

ProductVariantModel ()

Methods

checkoutUrl
(
  • quantity = 1
)
String
public

Get a checkout url for a specific product variant. You can optionally pass a quantity. If no quantity is passed then quantity will default to 1. The example below will grab a checkout url for 3 copies of the first variant:

const checkoutURL = product.variants[ 0 ].checkoutUrl(3);

Parameters:

  • [quantity = 1] Number optional

    quantity of variants

Returns:

String:

Checkout URL

Properties

available

Boolean

Variant in stock. Always true if inventory tracking is disabled.

compareAtPrice

String

Compare at price for variant. The compareAtPrice would be the price of the product previously before the product went on sale. For more info go here.

If no compareAtPrice is set then this value will be null. An example value: "5.00"

formattedPrice

String

Price of variant, formatted according to shop currency format string. For instance "$10.00"

grams

Number

Variant weight in grams. If no weight is defined grams will be 0.

id

String

Variant unique ID
